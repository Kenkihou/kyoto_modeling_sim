ローカル環境（PCのデスクトップ上）でテスト用のサーバーを起動し、ブラウザで動作確認できるようにするには、VS Codeのターミナルに以下のコマンドを入力します。

npm run dev

実行した後の手順
コマンドを入力して Enter を押すと、ターミナルに以下のようなURLが表示されます。

➜  Local:   http://localhost:5173/

表示された http://localhost:5173/ の文字を、Ctrl キーを押しながらクリックするか、ブラウザのURL欄に直接コピペして開きます。

ローカルサーバーを終了して入力を受け付ける状態に戻したいときは、ターミナル上で Ctrl + C キー を押し、バッチ ジョブを終了しますか (Y/N)? と聞かれたら y を入力して Enter を押してください。

これでデプロイ（Web公開）する前に、手元のパソコンだけで安全に新機能のテストや挙動の確認を行うことができます！
